# 🏭 ShelfBot — Autonomous Warehouse Forklift

Autonomous forklift robot for pick and place operations on warehouse shelves, built on ROS 2 and NVIDIA Isaac ROS.

---

## 🤖 Overview

ShelfBot is a warehouse automation project that enables a forklift robot to autonomously navigate, localise, and perform pick and place operations on shelves. Built using the NVIDIA Isaac ROS stack with a custom ROS 2 navigation pipeline.
